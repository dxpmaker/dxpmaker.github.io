一维数组

一维数组的意义在于定义一个能储存多的数据的保存空间
例：

        int main(){
            int a[4];//定义一个空间为四的数组（注意：在C语言中数组下标是从0开始，即a[4]没有意义且为空）
        return 0;
        }

对数组的定义也有多种办法
例：

        int main(){
            int a[4];
            int b[4]={1,2,3,4};//注意该方法如果赋值超过将会被警告
            int c[]={1,2,3,4};//该方法可以使程序自动判断需要赋值的数量，来决定数组大小
        return 0;
        }

数组的调用

        int main(){
            int n;
            int a[]={1,2,3,4,5,6,7};
            for(n=0;n<7;n++){
                printf("a[%d]:%d\n",n,a[n]);//遍历数组并输出
            }
        return 0;
        }

字符串的定义

一般而言在char型变量要输入大量的字符时一般的变量定义及调用极其复杂且低效，这时就需要用到char型数组
例：

        int main(){
            char a[]={"hello world"};
            printf("%s",a);//输出整个字符串（字符串即在C语言中指大量的字符）
        }

二维数组

相比于一维数组二维数组可以引入空间概念
如

相较于一维数组的直线二维数组进入了平面的范畴而其定义方式也有三种

        int main(){
            int a[2][3]={1,2,3,4,5,6};//自动分配到各个空间中
            int b[][2]={1,2,3,4,5,6}//只能省略行不能省略列
            int i,j;
            for(i=0;i<2;i++){
                for(j=0;j<3;j++){
                    printf("a[%d][%d]=%d\n",i,j,a[i][j]);
                }
            }
        return 0;
        }

多维数组

多维数组使用量小（一般会使用结构体更适合分类）但其使用与2维相似增加维度即可此处不在赘述。
