if判断语句

if判断

在C语言中if语句加上else有三种用法一种是if直接判断判断
栗子：

        if(a>b){//()中为逻辑判断但逻辑为真时其中的语句将会执行
            printf("a比b大");
        }

而这类方法存在局限性，比如我们需要真执行一种语句，而在为假时执行另一种语句。这时我们可以使用if eles进行判断
栗子：

        if(a>b){
            printf("a比b大");
        }eles{
            printf("b比a大或b等于a");
        }//当不满足条件时其将会执行

当然如果我们需要更多判断需求时我们也可以对if else连用

        if(a>b){
            printf("b比a小");
        }else if(a=b){
            printf("b等于a");
        }else if(a<b){//此处为方便理解写出
            printf("b比a大");
        }else{
            printf("错误");
        }

switch判断

switch在C语言中适合用于对某个变量判断的情况且这个数最好为整数（也可以为字符型或其他类型）
栗子： 

        int main(){
            int n=1;
            switch(n){                      //n为进行判断的变量
                case 1:printf("a=1");break;
                case 2:printf("a=2");break;
                case 3:printf("a=3");break;//判断条件后用:隔开
                default:printf("不满足判断条件");//该句的意义在于当不满足所有判断条件时在这里执行当前语句（相当于else）
            }
        }

在该判断语句中break的意义在于，跳出当前循环或跳出当前判断语句的其后所有判断

        int main(){
            int n=1;
            switch(n){
                case 1:
                case 3:
                case 5:
                case 7:
                case 9:printf("n为奇数");break;                  //当没有跳出命令时，不进行跳出，继续向下寻找
                case 2:
                case 4:
                case 6:
                case 8:printf("n为偶数");break;
                default:printf("不在1~9的范围内");
            }
        }
