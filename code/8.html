循环的概念

循环是指在特定情况下或指定的次数下重复执行同一代码或存在一定差别的代码行，有效的利用有助于减少代码数量

for循环

for循环是在C语言中极其常用的一个循环，其一般使用方式为
栗子：

        int main(){
            int i;
            for(i=1;i<5;i++){//for(;;)存在三个输入值，输入值一、代表进入循环前所执行的代码（有且仅执行一次），二、代表执行或继续循环代码所需要满足的条件，第、代表每次循环所需要满足的条件
                printf("%d",i);//%d代表输出（打印）整型数据并显示（具体会在输入输出同一解释）
            }//这样我们就能在屏幕上获得1234的结果
            return 0;
        }

while循环

while循环与for循环最大的不同之处在于，他是以提供的逻辑值为基础进行判断且循环本身不会自动增加较为简单

        int main(){
            int i=10,j=0;
            while(i!=j){//程序会在进入循环时判断，且每次运行到程序尾时返回程序头部进行判断。当逻辑值为真是（也就是1）程序会继续运行，为假(0)则跳出
                j++;
                i--;
            }
            printf("运行了%d次",i); 
            return 0;
        }
如果输入while(1)就相当于创造一个死循环，建议在适当的地方插入一个if(条件){break;}来跳出循环

do while循环

do while循环是while循环的变体其差别除了书写方式的差别也有书写地方的差别

        int main(){
            int i=11,j=0;
            do{
                i++;
                j+=2;
                printf("%d %d\n",i,j);//\n代表换行即每次输出都将换一次行
            }while(i!=j&&j!=0);//判断条件在句尾代表着当程序运行到这里时才判断（可以理解为无论如何程序都将运行一次）为假跳出为真则继续。(&&代表与逻辑)
            return 0;
        }
