文件指针的定义与应用

在C语言中文件的读写是依靠指针来进行的
栗子

        int main(){
            FILE *fp1 =fopen("文本.txt","w");
            //fopen中的输入分别为  文档地址（此处用的是相对地址），打开方式(此处使用仅写)
            if(fp1==NULL){
                puts("错误,文件不存在或无法打开");
            }
            fputs("Hello world",fp1);//其实就是输入输出函数最前面加个f就不再细讲了(此处为写入字符串)
            fclose(fp1);//关闭文件指针（相当于保存）（请注意关闭否则将会浪费内存）
            return 0;
        }
    
        int main(){
            FILE *fp1=fopen("文本.txt","r");
            char A[100];//定义一个用于保存的字符串数组
            fgets(A,100,fp1);//读取文件流需定义一个保存文本的字符串也就是输入值一，同时第二个输入值为订意读取文件的大小（文本指针会停在读取结束的地方），第三个值为文件指针
            puts(A);//输出文本内容
            fclose(fp1);//关闭文件
        }
文本操作命令


命令 操作 区别 文件要求及操作
r 读 从文件头开始 文件需存在，否则将打开失败
r+ 读写 从文件头开始 文件需存在，否则将打开失败
w 写 从文件头开始 文件不存在则创建，存在则清空
w+ 读写 从文件头开始 文件不存在则创建，存在则清空
a 写 从文件尾开始 文件不存在进行创建，存在则从文件尾继续写入
a+ 读写 从文件头读取，从文件尾写入 文件不存在进行创建，存在则从文件尾继续写入
