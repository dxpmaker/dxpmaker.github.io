变量

一般情况下可变的量被称为变量、在C语言中没有使用static定义为常量的都为变量。

常量

常量一般指不可变的量也就是静态变量。其在程序的生命周期中仅能被定义或被赋值一次。任何多余的赋值都可能导致编译错误或程序崩溃（也存在常量拒绝访问）。使用static将变量定义为常量 

            static int a=1;//定义一个静态变量

修饰语句

在程序中与常量定义类似的定义如long（长定义）可用来定义长整型、长双精度等，而short（短定义）也可定义短整型也存在unsigned（无符号即仅为正）的符号来获取定义拒绝接受负值定义的变量且能延长正值的范围（能与long、short)同时定义 

            unsigned int a;//定义一个无符号整型
            long int b;//定义一个长整型
            short int c;//定义一个短整型
            unsigend long int d;//定义一个无符号长整型
