算术运算符

算数运算符在程序中是很常见的一种运算符。包括但不限于+-*/=%再内的计算（%为取余计算符 5%2 为5除2取余即为1）应用范围大。同时不同类型的变量能共同进行计算但结果只会输出精度最高的类型(计算顺序符合数学运算逻辑)

注，因单目运算符无意义在此写出单目运算符仅计算双目运算符
名称 符号 示例
加法 + a=a+b
减法 - a=a-b
取模（余） % a=5%2 //(a==1)
乘法 * a=a*b
除法 / a=a/b

自增自减运算符

该运算符为C语言特有的运算符（部分语言可能也有）作用在于似变量或数+1或-1

--a; ++a; a--; a++;//运算符放于前后无影响前后影响

赋值运算符

该运算符与自增运算符类似但又有所不同，其应用范围更广。使用该类运算符可有效减少代码量

符号 名称 等价于
等于 = NULL
加等于 += a=a+b
减等于 -= a=a-b
乘等于 *= a=a*b
除等于 /= a=a/b
模等于 %= a=a%b

关系运算符

关系运算符为比较大小判断两个数的大小其将返回1或0（真或假）来判断

符号 功能 符号 功能
> 大于 <= 小于等于
>= 大于等于 == 等于（注存在两个等号，单个将会变成赋值）
< 小于 != 不等于

逻辑表达式

逻辑运算符用于其在程序进行逻辑条件下的判断，即一、与、或、非三者进行运算

符号 功能
&& 逻辑与（且）
|| 逻辑或
! 单目逻辑非

位逻辑运算符

在C语言中位逻辑运算符的意义在于其能对底层逻辑进行修改以达到部分汇编语言的效果（当然不深究的就不要看这部分了汇编语言比高级语言还要让人掉发）

符号 功能
& 位逻辑与
| 位逻辑或
^ 位逻辑非
~ 取补
