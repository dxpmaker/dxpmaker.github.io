输出函数

输出函数有

        printf();//标准输出流函数函数
        puts();//标准字符串输出函数
        putc();//字符输出到指定的文件（宏，较复杂不细讲）（需要指定输入的值）
        putchar();//输出一个字符（在多次调用会从上次的地方继续输出）

在输出函数中printf是基础输出能输出字符整型等等而puts则使用较少一般用于输出指定的字符串（代码小）putchar则是用于控制输出量

        int main(){
            char a[]={"Hello world"};
            int i;
            printf("%s\n",a);
            puts("HelloWorld\n");
            puts(a);//也可如此使用
            for(i=5;i<5;i++){;
                putchar(a);
            }
            return 0;
        }

输入函数

输入函数有

        scanf();//与printf使用方法类似差别在于其在获取错误输入时可能会出现错误（%d时输出英文会导致程序无法结束）
        gets(); //获取输入的字符串
        getc();//获取的字符从指定的文件中（宏，较复杂不细讲）（需要指定输入的值）
        getchar();//获取键盘的输入

在输入函数中scanf为标准的输入函数获取键盘所有输入gets则是获取输入的字符串getchar则会每次读取一个字符并将指针想后移动一次

        int main(){
            char a[10];
            char b;
            scanf("%s",a);//也可换成gets(a);//注意仅能存在一个全输入点（好像是因为stdio.h的输入流问题）
            puts(a);
            b=getchar();
            puts(b);
            return 0;
        }
符号 作用
%d 整型输出
%ld 长整型输出
%o 以八进制数形式输出整数
%x 以十六进制数形式输出整数
%u 以十进制数输出unsigned型数据(无符号数)
%c 用来输出一个字符
%s 用来输出一个字符串
%f 用来输出实数，以小数形式输出
%e 以指数形式输出实数
%g 根据大小自动选f格式或e格式，且不输出无意义的零
