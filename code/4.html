在高级语言中几乎都存在着常变量的声明与定义。其差别在于对不同的基础变量定义的方式、支持的类型有所不同

变量


变量顾名思义就是可以被程序改变的量、在其基础上、不能被改变的量称为常量（一般表现为赋值即会导致程序错误）


在C语言中变量的种类不多仅有int（整数型）、char（字符型）、float（单精度型/浮点型）、double（双精度型）


int类型


int类型即为整型其能接受-2147483648~2147483647的取值范围。注仅能存储整数的量。
若赋值如1.2形式的单精度小数时会导致精度改变，其在内存中占4个字节



        int a;     //声明一个整型变量
        int a=10;  //声明变量的同时进行赋值
        //注意不能再声明变量时不能将数字、符号放于名称的第一位。同时不能带有-连接符应使用_下划线进行单词连接

char


char型函数在于其的能保存字符注意是字符（即a,b,c,d称为字即其字节大小仅为1，中文在不同的编码下存在不同的字节大小）
其能保存文字并输出，同时因为ASCII码为基础的性质，导致字符型常量能接受整数赋值输出对应ASCII码的字符，其在内存中占1个字节



        char a;         //声明一个char变量
        char a = 'b';   //赋值使用'进行赋值字符而字符串才使用"进行赋值
        char a = 0;     //在ASCII码中0代表空格

float，与double


float型为单精度型。double型为双精度型,两个变量都可称为实型变量（不太好区分）其最大的差别在于其精度的差异以及在内存中占的字节数。
float在内存中占4个字节取值范围在 -3.4E+38 ~ 3.4E+38 (注意此处使用的是16进制计算单位)而double在内存中占8字节其取值范围为负值-1.7976E+308 ~ -4.94065645841246544E-324
正值4.94065645841246544E-324 ~ 1.797693E+308（留个印象即可这两个类型在正常情况下使用甚少）


auto


在C语言中auto并不代表一种变量类型而是代表自动的变量。即每次运行到该变量时程序会对其进行重新定义并使其初始化
