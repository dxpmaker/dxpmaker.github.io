函数的嵌套

函数的返回值中不仅仅可以用来作为返回函数也可以用来调用自身或其他函数

        int main(){
            one();
            return 0;
        }
        void one(){
            printf("函数一");
            return  tow();//调用其他函数
        }
        void tow(){
            printf("函数二");
            return ;
        }

自我调用类似于循环需要。注意:调用需要个if或其他函数来一return;否则将会死循环

        int main(){
            printf("%d",one(1));
            return 0;
        }
        int one(int a){
            if(a==10)return a;//当if条件满足时将a返回
            a++;    
        return one(a);
        }

局部变量与全局变量

一般而言我们在函数类定义的就是局部变量。该变量仅在当前函数有效（子函数不能调用主函数的变量，主函数一样不能调用子函数的变量）。而全局变量不同其能在本程序中被任意函数调用赋值，不受函数范围的影响。 

        int a;//主函数的定义
        int main(){
            one();
            printf("%d",a);//在主程序中调用全局变量
        }
        void one(){
            a=10;//在子程序中调用全局变量
        }
